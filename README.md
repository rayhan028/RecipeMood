# RecipeMood

A mood-based recipe recommendation web app built with ASP.NET Core MVC.  
Log how you feel, record mood intensity and notes, and get recipe suggestions that match your emotions.  
View stats, track mood trends, and export your journal to JSON.

## Features
- Log moods with intensity and notes  
- Get personalized recipe suggestions  
- View mood and recipe statistics  
- Simple trend visualization  
- JSON export of mood logs  

## Tech Stack
- ASP.NET Core MVC (C#)  
- Razor Pages  
- In-memory data storage
